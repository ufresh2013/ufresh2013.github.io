Google-tag-manager设置

2018-08-16

1. 登录

进入Google tag官网

2. 新建容器

一个账号可拥有多个容器，一个容器放一个广告，页面左上角可切换容器。

  • 进入跟踪代码 管理tab
  • 点击右上角 + 创建容器
  • 设置容器名称和容器使用位置

3. 新建代码

页面添加<div id="vip"></div>，代码类型选择自定义HTML，通过js设置#vip的innerHTML

  • 设置触发器: All Pages 网页浏览

4. 提交发布

  • 安装Google跟踪管理器: 将代码复制到你的网站上。
    关闭调试url上添加&gtm_debug=
    修改f=document.getElmentById('vip'),f.appendChild(j,f)

  • 提交: 填写版本名称，版本说明

  • 预览
  • 发布
