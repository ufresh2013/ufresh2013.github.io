因为内置chrome内核，所以开发者可以用html/css/js来构建页面，这一部分运行在渲染进程中。
因为内置nodejs环境，所以可以访问计算机本地的资源：读写磁盘文件、创建进程、本地通知，这一部分运行在主进程中。
